Hadoop 0.19.1 Release Notes

Hadoop 0.19.1 fixes serveral problems that may lead to data loss from the file system and makes some incompatible changes from Hadoop 0.19.0. For instance, the file append API has been disabled in this release due to implementation issues that can lead to data loss. The bug fixes are listed below.

Hadoop 0.19.0 Release Notes
