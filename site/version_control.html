Overview

The Hadoop source code resides in the Apache Subversion (SVN) repository. The command-line SVN client can be obtained here. The TortoiseSVN GUI client for Windows can be obtained here. There are also SVN plugins available for both Eclipse and IntelliJ IDEA.

Web Access (read-only)

The source code can be browsed via the Web at http://svn.apache.org/viewcvs.cgi/lucene/hadoop/. No SVN client software is required.

Anonymous Access (read-only)

The SVN URL for anonymous users is http://svn.apache.org/repos/asf/lucene/hadoop/. Instructions for anonymous SVN access are here.

Committer Access (read-write)

The SVN URL for committers is https://svn.apache.org/repos/asf/lucene/hadoop/. Instructions for committer SVN access are here.
